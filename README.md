# Lab 6: Weather Station Time Series Analysis (ATOC 5815)

## Overview
This project analyzes weather station data from the CU Boulder ATOC station using **Python and pandas**. The goal is to explore time series data and apply key data analysis techniques.

## Key Features
- Time-indexed weather data using pandas
- Temperature and dew point analysis
- Relative humidity trends
- Wind speed and peak gust comparisons
- Solar radiation patterns
- Rainfall (instantaneous and cumulative)
- Temperature anomalies from daily mean
- 24-hour rolling averages

## Methods Used
- Pandas DataFrames with datetime indexing
- Boolean filtering
- Rolling window analysis
- Resampling and aggregation
- Time series plotting with matplotlib

## Output
The notebook generates a multi-panel figure showing:
- Temperature, dew point, and rolling mean
- Humidity trends
- Wind behavior
- Solar radiation
- Rainfall accumulation
- Temperature anomalies
